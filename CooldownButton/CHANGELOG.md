# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] – 2026-06-30

### Added
- Initial release.
- `CooldownButton` MonoBehaviour — ability-style button with radial sweep cooldown.
- Configurable cooldown duration, unscaled time support, and auto-disable toggle.
- Radial fill overlay with configurable fill method, origin, and direction.
- Optional TMP countdown label showing remaining seconds.
- `StartCooldown()` and `ResetCooldown()` scripting API.
- `OnClick`, `OnCooldownStart`, `OnReady`, and `OnCooldownProgress` UnityEvents.
