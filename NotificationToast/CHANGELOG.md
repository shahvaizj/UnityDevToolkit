# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] – 2026-06-25

### Added
- Initial release.
- NotificationToast component with queue-based notification display.
- Four severity levels: Info, Success, Warning, Error.
- Per-severity styling: background color, text color, and optional icon sprite.
- Slide animation from configurable direction (Top, Bottom, Left, Right) with cubic easing.
- Configurable fade-in, display, and fade-out durations.
- Unscaled time support for paused-game notifications.
- OnToastShown and OnToastDismissed UnityEvents.
- DismissCurrent, DismissAll, and ClearQueue API methods.
