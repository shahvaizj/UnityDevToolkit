# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] – 2026-06-21

### Added
- Initial release.
- SmoothNumText component with AnimationCurve-driven number interpolation.
- Configurable prefix, suffix, decimal places, and thousands separator formatting.
- AnimateTo, AddValue, SetValue, and SnapToTarget API plus a static Instance accessor.
- Scaled / unscaled time support.
- OnAnimationStarted, OnAnimationComplete, and OnValueChanged UnityEvents.
