# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] – 2026-07-11

### Added
- Initial release.
- ScriptableVariable<T> abstract base with runtime Value, Initial Value, and Reset On Play support.
- FloatVariable with Add helper and OnValueChanged UnityEvent.
- IntVariable with Add helper and OnValueChanged UnityEvent.
- BoolVariable with Toggle helper and OnValueChanged UnityEvent.
- StringVariable with OnValueChanged UnityEvent.
- ResetValue() to manually restore the initial value on any variable type.
