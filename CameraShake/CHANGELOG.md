# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] – 2026-06-16

### Added
- Initial release.
- CameraShake component with a trauma-based, Perlin-noise driven shake.
- Configurable per-axis translation and rotation amplitudes, trauma decay, exponent, and noise frequency.
- Scaled / unscaled time support.
- AddTrauma, Shake, SetTrauma, and StopImmediate API plus a static Instance accessor.
- OnShakeStarted and OnShakeStopped UnityEvents.
