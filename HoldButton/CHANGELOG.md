# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] – 2026-06-04

### Added
- Initial release.
- HoldButton component with configurable hold duration and progress tracking.
- OnHoldComplete, OnHoldCancel, OnHoldProgress UnityEvents.
- Visual feedback via target Graphic or Renderer with fill/color lerp.
- Interaction threshold for early cancel.
