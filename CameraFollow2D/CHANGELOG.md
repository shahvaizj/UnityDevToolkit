# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] – 2026-07-06

### Added
- Initial release.
- CameraFollow2D component with SmoothDamp-based follow and configurable deadzone.
- Optional world-bounds clamping that accounts for orthographic viewport size.
- SetTarget, SetBounds, and SnapToTarget scripting API.
- Unscaled time support for following while paused.
- Deadzone and bounds gizmos drawn in the Scene view when selected.
