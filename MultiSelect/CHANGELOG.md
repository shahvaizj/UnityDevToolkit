# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] – 2026-06-20

### Added
- Initial release.
- MultiSelect component with left/right arrow buttons cycling a serialized string list.
- CurSelected (index) and Value (text) accessors, populated on Awake.
- Optional wrap-around with automatic arrow disabling at the ends.
- SetSelected, SetValue, and SetOptions runtime API.
- OnSelectionChanged and OnValueChanged UnityEvents.
