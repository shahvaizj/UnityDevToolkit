# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] – 2026-06-27

### Added
- Initial release.
- TooltipSystem component with smart screen-edge repositioning.
- TooltipTrigger component for per-element hover and long-press tooltips.
- Configurable hover delay, fade-in/fade-out durations, and pointer offset.
- Automatic pivot flipping when tooltip would clip off-screen.
- Long-press support with movement tolerance for touch devices.
- Pointer-following mode for cursor-tracking tooltips.
- OnTooltipShown and OnTooltipHidden UnityEvents.
