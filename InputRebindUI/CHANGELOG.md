# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] – 2026-07-10

### Added
- Initial release.
- RebindActionUI component for interactive per-row control rebinding, with excluded controls, cancel control, and start/complete/cancel UnityEvents.
- InputRebindManager component to save, load, and reset all binding overrides via PlayerPrefs.
