# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] – 2026-07-08

### Added
- Initial release.
- ParallaxLayer component with per-axis parallax factor driven by a camera transform.
- Optional seamless infinite horizontal/vertical scrolling for tiled sprites.
- Auto-detection of Camera.main when no camera is assigned.
- SetCamera and ResetLayer scripting API for runtime camera swaps and level transitions.
