# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] – 2026-07-22

### Added
- Initial release.
- OrbitShowcaseCamera component with idle auto-rotate, drag-to-orbit, and scroll/pinch zoom.
- Configurable pitch and distance clamping with smoothed yaw/pitch/distance transitions.
- OnDragStarted, OnDragEnded UnityEvents.
- SetTarget and ResetView API for runtime target swaps between showcased items.
