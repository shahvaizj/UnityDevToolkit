# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] – 2026-06-20

### Added
- Initial release.
- ScreenFlash component with AnimationCurve-driven full-screen color overlay.
- Configurable default color, duration, and alpha curve.
- Multiple Flash() overloads for color, duration, and curve control.
- OverlapMode enum: Restart, Ignore, or Enqueue overlapping flashes.
- Scaled / unscaled time support.
- Static Instance accessor for one-line triggers.
- OnFlashStarted and OnFlashComplete UnityEvents.
