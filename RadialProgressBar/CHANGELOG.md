# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] – 2026-06-29

### Added
- Initial release.
- RadialProgressBar component with 0–1 fill API.
- Gradient color support that shifts with progress value.
- Smooth animated transitions with configurable speed.
- Configurable fill method, origin, and clockwise direction.
- OnProgressChanged, OnFillComplete, and OnFillReset UnityEvents.
- SetProgressImmediate and IncrementProgress helper methods.
